Soal
1. Deskripsi Tugas
Mahasiswa diminta untuk membangun sebuah aplikasi berbasis Web Service (REST API) yang
mengintegrasikan kecerdasan buatan (AI). Aplikasi harus mencakup fungsionalitas manajemen data dasar
(CRUD) dan fitur cerdas menggunakan layanan pihak ketiga.
2. Opsi Tema Proyek
Pilih salah satu dari tema berikut atau gunakan rekomendasi tambahan:
1. Sistem Manajemen Perpustakaan: Pengelolaan buku dan peminjaman.
2. Sistem Reservasi Tiket: Pemesanan tiket transportasi atau acara.
3. Sistem Transaksi E-Wallet: Simulasi top-up dan pembayaran digital.
4. Rekomendasi Lainnya:
✓ Smart Inventory Tracker: Manajemen stok barang dengan analisis otomatis.
✓ Healthcare Appointment: Sistem pendaftaran pasien ke dokter.
✓ Personal Finance Manager: Pencatatan pengeluaran dan pemasukan.
✓ Recipe & Meal Planner: Pengelolaan resep masakan.
3. Persyaratan Teknis
• Fungsionalitas: Aplikasi harus berjalan dengan baik (functional).
• CRUD API: Implementasikan minimal 3 operasi CRUD (Contoh: Create, Read, Delete).
• Integrasi AI: Wajib mengintegrasikan Groq API (misal: untuk chatbot, ringkasan data, atau
rekomendasi otomatis).
• Dokumentasi: Membuat dokumentasi API menggunakan Postman.
4. Komponen Pengumpulan
1. Repository GitHub: Tautan kode sumber proyek.
2. Laporan Dokumentasi (PDF):
✓ Tangkapan layar (Screenshot) antarmuka aplikasi.
✓ Penjelasan detail mengenai fitur-fitur yang tersedia.
✓ Dokumentasi endpoint API (Method, URL, Request Body, Response).
